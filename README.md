# Transport Car - Web Application

A responsive web application for a transport/car service platform. This project includes user authentication, product management, and informational pages built with HTML, CSS, and JavaScript.

## Features

- **User Authentication**: Login functionality with secure user management
- **Product Showcase**: Display and manage transport/car services
- **Multiple Pages**: Home, About, Contact, Product, and Strategy pages
- **Responsive Design**: CSS-based styling for various screen sizes
- **User Management**: Backend functionality for managing users

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side installation required for basic functionality

### Usage

1. **Open the Application**
   - Open `index.html` in your web browser to view the main page
   - Navigate using the menu to access different sections

2. **Login**
   - Click on the login link
   - Use `login.html` to access user authentication

3. **Browse Pages**
   - **Home**: Main landing page with overview
   - **About**: Company/service information
   - **Product**: Available transport/car services
   - **Contact**: Contact information and form
   - **Strategy**: Business strategy and goals

## Development

### To Modify Styling
- Edit `css/style.css` for global changes
- Edit `css/about.css` for About page specific changes

### To Add New Features
- Update relevant JavaScript files in the `js/` folder
- Create new HTML files in the `pages/` folder if needed
- Add corresponding styles to CSS files
